💡 HOW IT HELPS YOU!!

Understand the purpose and usage of this Breast Cancer Diagnosis Dashboard:

  • 🔍 Purpose: This tool helps in predicting whether a tumor is Benign or Malignant using a logistic regression model trained on breast cancer data.
  • 👩‍⚕️ Who Can Use: Medical students, data analysts, healthcare researchers, or anyone exploring machine learning for healthcare.
  • 📂 Required Data Format: CSV file with the diagnosis column (M for malignant, B for benign), along with numerical features like radius, texture, area, smoothness, etc.
  • 📈 What It Does: Performs classification, shows ROC curve, sigmoid, confusion matrix, class distribution, correlation, and top 10 feature importances — with live visuals.
  • 📌 Rules for Upload:
    • Make sure your file is in .csv format
    • It must include a diagnosis column
    • Remove missing values before uploading
  • 📤 Output: Easy-to-understand metrics + visuals + data preview
  • 🌓 Bonus: You can toggle between dark and light mode anytime
